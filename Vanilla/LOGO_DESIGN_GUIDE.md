# Logo Design Guide for Permanent Beauty

## Current Website Color Scheme
- **Primary Red**: #C41E3A (Deep elegant red)
- **Secondary Black**: #000000 (Classic black)
- **Accent White**: #FFFFFF (Clean white)
- **Light Gray**: #F5F5F5 (Subtle background)

## Logo Design Specifications

### Recommended Logo Style
- **Type**: Wordmark + Icon or Wordmark only
- **Style**: Elegant, modern, sophisticated
- **Mood**: Professional, luxurious, trustworthy

### Typography Recommendations

#### Primary Font Options (for "Permanent Beauty"):
1. **Playfair Display** - Elegant serif, luxury feel
2. **Cormorant Garamond** - Classic, refined
3. **Cinzel** - Sophisticated, modern classic
4. **Bodoni** - High-fashion, timeless
5. **Libre Baskerville** - Professional, elegant

#### Font Styling:
- **"Permanent"**: 
  - Font weight: 300-400 (Light/Regular)
  - Color: #000000 (Black)
  - Letter spacing: 0.05em

- **"Beauty"**: 
  - Font weight: 600-700 (Semi-bold/Bold)
  - Color: #C41E3A (Red)
  - Letter spacing: 0.1em

### Icon/Symbol Ideas (if using)
- Minimalist lipstick outline
- Abstract eyebrow arc
- Elegant makeup brush
- Stylized beauty mark or beauty-related iconography
- Geometric heart shape
- Delicate floral element

### Layout Options

**Option 1: Stacked**
```
PERMANENT
BEAUTY
```

**Option 2: Side by Side**
```
PERMANENT BEAUTY
```

**Option 3: With Icon**
```
[Icon] PERMANENT
       BEAUTY
```

### Color Variations Needed

1. **Primary Version** (for light backgrounds)
   - Black + Red

2. **Inverted Version** (for dark backgrounds/header)
   - White + Red

3. **Monochrome White** (for dark red backgrounds)
   - All white

4. **Monochrome Black** (for light backgrounds)
   - All black

## DIY Logo Creation Tools

### Free Online Tools:

1. **Canva** (https://canva.com)
   - Free templates
   - Easy to use
   - Export as PNG with transparency

2. **LogoMaker** (https://logomaker.com)
   - Simple interface
   - Professional results

3. **Hatchful by Shopify** (https://hatchful.shopify.com)
   - Free
   - Beauty industry templates

4. **Looka** (https://looka.com)
   - AI-powered
   - Professional quality

### Design Steps:

1. **Choose a font** from the recommendations above
2. **Set colors**:
   - "PERMANENT" in black (#000000)
   - "BEAUTY" in red (#C41E3A)
3. **Add spacing** between letters for elegance
4. **Optional**: Add a simple icon/symbol
5. **Export** as PNG with transparent background
6. **Create versions**:
   - Width: 400-600px
   - Height: 80-120px
   - File format: PNG with transparency
